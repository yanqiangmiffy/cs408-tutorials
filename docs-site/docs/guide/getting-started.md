# 快速开始

欢迎使用408数据结构Python实现文档！本项目针对计算机考研408科目中的数据结构部分，提供了完整的Python代码实现、原理讲解和考研要点总结。

## 📚 内容结构

本教程严格按照王道408考研大纲编排，共8个章节：

| 章节 | 内容 | 重点 |
|------|------|------|
| 第1章 | 绪论 | 时间复杂度分析 |
| 第2章 | 线性表 | 顺序表、链表 |
| 第3章 | 栈与队列 | 栈的应用、循环队列 |
| 第4章 | 串 | KMP算法 |
| 第5章 | 树与二叉树 | 遍历、哈夫曼树、并查集 |
| 第6章 | 图 | BFS/DFS、最小生成树、最短路径、拓扑排序 |
| 第7章 | 查找 | BST、AVL树、散列表 |
| 第8章 | 排序 | 9种排序算法实现与对比 |

## 🚀 本地运行

如果你想在本地运行这个文档站点：

```bash
# 进入文档目录
cd docs-site

# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build
```

## 💡 使用建议

1. **基础薄弱**：按章节顺序学习，先理解原理，再看代码实现
2. **考研复习**：重点关注每个章节的「考研要点&易错点」和「复杂度总结」
3. **编程练习**：看完代码后关闭文档自己手写实现一遍
