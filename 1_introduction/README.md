## 考研要点

- **数据结构三要素**: 逻辑结构 + 存储结构 + 数据运算
- **逻辑结构**: 集合、线性、树形、图状
- **存储结构**: 顺序、链式、索引、散列
- **时间复杂度**: O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2^n)
- **空间复杂度**: 递归算法 = O(递归深度)
