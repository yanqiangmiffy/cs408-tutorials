# 第1章 绪论

> **考研要点速记**：数据结构三要素是逻辑结构、存储结构、运算；时间复杂度分析是每年必考题，重点掌握递推式求解和大O记号表示。

## 1. 概念介绍 <a id="concepts"></a>

### 1.1 基本概念

- **数据**：描述客观事物的符号，是计算机可以操作的对象
- **数据元素**：数据的基本单位，通常作为一个整体进行考虑和处理
- **数据项**：构成数据元素的不可分割的最小单位
- **数据对象**：性质相同的数据元素的集合，是数据的一个子集

### 1.2 数据结构三要素

#### 逻辑结构
指数据元素之间的逻辑关系，与数据的存储无关：
- **线性结构**：一对一关系（线性表、栈、队列、串）
- **非线性结构**：
  - 集合：元素之间除了"同属于一个集合"外没有其他关系
  - 树形结构：一对多关系
  - 图形结构：多对多关系

#### 存储结构（物理结构）
指数据结构在计算机中的表示（映像）：
- **顺序存储**：把逻辑上相邻的元素存储在物理位置上也相邻的存储单元里
- **链式存储**：不要求逻辑上相邻的元素物理位置也相邻，通过指针表示元素之间的逻辑关系
- **索引存储**：在存储元素信息的同时，还建立附加的索引表
- **散列存储**：根据元素的关键字直接计算出该元素的存储地址

#### 运算
施加在数据上的操作，包括运算的定义和实现：
- 定义是针对逻辑结构的，指出运算的功能
- 实现是针对存储结构的，指出运算的具体操作步骤

## 2. 时间复杂度 <a id="time-complexity"></a>

### 2.1 大O记号定义

算法的时间复杂度是指算法运行所需要的时间，通常用大O记号表示：
- T(n) = O(f(n)) 表示存在正常数c和n₀，使得当n ≥ n₀时，0 ≤ T(n) ≤ c·f(n)
- 表示算法时间复杂度的上界

### 2.2 常见的时间复杂度排序

从小到大依次为：
**O(1) < O(logn) < O(n) < O(nlogn) < O(n²) < O(n³) < O(2ⁿ) < O(n!) < O(nⁿ)**

### 2.3 时间复杂度分析方法

#### 循环嵌套分析
- 最内层循环的执行次数是时间复杂度的关键
- 嵌套循环的复杂度是各层循环复杂度的乘积

**示例**：
```python
n = 100
for i in range(n):          # 执行n次
    for j in range(n):      # 执行n次
        print(i + j)        # O(1)
```
时间复杂度：O(n²)

#### 递推式求解
对于递归算法，可以通过递推式求解时间复杂度：

**示例：斐波那契数列**
```python
def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)
```
递推式：T(n) = T(n-1) + T(n-2) + O(1)
时间复杂度：O(2ⁿ)

## 3. 空间复杂度 <a id="space-complexity"></a>

### 3.1 定义

算法的空间复杂度是指算法运行所需要的内存空间，用S(n) = O(g(n))表示。

### 3.2 原地算法
如果算法所需的辅助空间为常量，即O(1)，则称该算法为**原地算法**。

**示例**：交换两个变量的值
```python
def swap(a, b):
    temp = a    # 辅助空间O(1)
    a = b
    b = temp
    return a, b
```
空间复杂度：O(1)，属于原地算法。

## 3. 考研重点 & 易错点

- ⚠️ 易错点：混淆时间复杂度的上界、下界和紧界，考研只要求掌握大O表示的上界
- 📌 高频考点：递归算法的时间复杂度分析，递推式求解
- ⚠️ 易错点：计算时间复杂度时忽略常数项和低次项，只保留最高次项且系数为1
- 📌 高频考点：常见算法的时间复杂度记忆（二分查找O(logn)，快速排序平均O(nlogn)等）

## 4. 复杂度总结表

| 常见算法 | 时间复杂度 | 空间复杂度 |
|----------|-----------|-----------|
| 简单循环 | O(n) | O(1) |
| 二分查找 | O(logn) | O(1) |
| 冒泡/插入/选择排序 | O(n²) | O(1) |
| 快速排序（平均） | O(nlogn) | O(logn) |
| 归并排序 | O(nlogn) | O(n) |
| 斐波那契递归 | O(2ⁿ) | O(n) |
