# 第11章 冲刺与速记

> 这一章不再讲“完整知识体系”，而是专门服务于 408 数据结构备考后期。内容覆盖高频考点速记、手写模板、易错点、复杂度总表、真题题型索引、答题模板、30 天计划、自测题、一页纸总结和错题复盘模板。

---

## 使用顺序

1. 先看 [高频考点速记](01-high-frequency.md)，把 8 章主线再压缩一遍。
2. 再看 [手写模板库](02-handwriting-templates.md) 和 [大题答题模板](06-answer-templates.md)，训练考场表达。
3. 配合 [易错点总表](03-mistakes.md) 和 [复杂度对比总表](04-complexity-comparison.md) 查漏补缺。
4. 用 [真题题型索引](05-qa-taxonomy.md) 把年份题和经典模型连起来。
5. 最后按照 [30 天复习路线](07-roadmap-30days.md) 与 [章节自测题](09-self-test.md) 执行冲刺。

## 建议搭配

- 主线不牢时：优先回看第 2、5、6、7 章正文，再配合本章速记。

- 模拟波动大时：重点看易错点、错题复盘模板和 30 天路线。
