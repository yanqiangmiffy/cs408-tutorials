# 错题复盘模板

## 使用方法

- 每次模拟或刷题后，只记录真正会影响得分的问题。
- 不要只写“粗心”，要写清楚是哪一种粗心。

## 模板 1：单题复盘

```text
题目：
日期：
来源：

我原来的错误：

正确思路：

我为什么会错：
1.
2.
3.

这个错误属于哪一类：
- 概念不清
- 模板不熟
- 边界漏掉
- 复杂度判断错误
- 书写表达不清

下次看到什么特征时，应该立刻想到这题：

我现在的最终模板：
```

## 模板 2：章节复盘

```text
章节：

最不稳的 3 个点：
1.
2.
3.

最常丢分的题型：
1.
2.
3.

需要再手写的模板：
1.
2.
3.
```

## 模板 3：模拟卷复盘

```text
模拟日期：
总分：

选择题失分原因：

大题失分原因：

本次暴露出的核心短板：

接下来 3 天要补什么：
1.
2.
3.
```

## 建议

- 错题本页数不必多，但每条都要能指导下一次复习。
- 同一个错误如果连续出现两次，必须回到主线正文重新学。
