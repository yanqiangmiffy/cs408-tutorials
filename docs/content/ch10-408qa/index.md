## 使用建议

1. 先完成主线 8 章，再回到本章按年份刷真题，效果最好。
2. 每题优先抓“考点本质”和“标准做法”，不要被题面表述带偏。
3. 把相似题一起做，训练把真题抽象成经典算法模型的能力。
4. 高难度年份建议重点复盘：2011、2018、2020。
